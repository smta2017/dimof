## Dimofinf Test installation

copy .env.example to .env file

then

1 - `composer update`

2 - `php artisan install:app`

3 - `npm i && npm run dev`

Now you can access your app throw `http://127.0.0.1:8000`


### Test

To run unit test

` ./vendor/bin/phpunit `
